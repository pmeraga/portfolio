# Engineering Portfolio
Collection of past/current projects

CNN+LSTM - an ongoing research project with Northwestern University's Wing Kam Liu Group on developing an efficient classification model for keyhole deformations in Laser Bed Powder Fusion(LBPF)

NOVA_espidf - a collection of embedded systems files in C for programming a differential wheeled-robot with real-time video feed and object avoidance

CE_93_project - a statistical study on the correlation between a country's % access to safe water and their citizen's life expectancy over time for Civil Engineering 93: Engineering Data Analytics

K_Means_Clustering_Asteroid_Data - a previous research project under the Wing Kam Liu Group that utilized K-Means Clustering to traverse and sort NASA's Asteroid Watch Database based on collision probability and potential impact capabilities
